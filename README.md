## Instrucciones de Uso de la Aplicación

1. **Acceso al sistema**

   * Abrir el navegador web
   * Ingresar a la URL:

   ```
   http://localhost:8080/<nombre-del-proyecto>/productos
   ```

2. **Visualizar productos**

   * Al ingresar, se mostrará la lista de productos disponibles
   * Se visualizan: ID, nombre, categoría, precio y stock

3. **Registrar un nuevo producto**

   * Hacer clic en **“+ Nuevo Producto”**
   * Completar el formulario con los datos requeridos
   * Presionar **“Guardar”**
   * El producto aparecerá en la lista

4. **Editar un producto**

   * Hacer clic en **“Editar”** en el producto deseado
   * Modificar los campos necesarios
   * Presionar **“Actualizar”**
   * Los cambios se reflejan en la lista

5. **Eliminar un producto**

   * Hacer clic en **“Eliminar”**
   * Confirmar la acción en el cuadro emergente
   * El producto será eliminado de la lista

6. **Mensajes del sistema**

   * Después de cada acción, se mostrará un mensaje indicando si la operación fue exitosa

7. **Manejo de errores**

   * Si ocurre algún error, se mostrará una página informativa indicando el problema
   * Desde allí se puede regresar al inicio del sistema
